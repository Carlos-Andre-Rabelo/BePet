🐾 BePet – Projeto Acadêmico

Este repositório contém o desenvolvimento de uma aplicação web fictícia chamada BePet, criada como exercício prático de programação web.
O projeto é composto por uma landing page para apresentação da loja e um painel administrativo para gerenciar mensagens recebidas via formulário de contato.

👤 Usuário (Landing Page)

-- Exibição da landing page da loja fictícia BePet.
-- Formulário de contato para envio de mensagens.
-- Estilização responsiva e moderna (cores, tipografia, efeitos).

🔑 Administrador

-- Acesso à tela administrativa.
-- Visualização de mensagens enviadas via formulário (nome, e-mail e conteúdo).
-- Exclusão de mensagens diretamente da interface, com integração ao banco de dados.
-- Interface estilizada para manter consistência com a identidade visual do projeto.

🛠️ Tecnologias Utilizadas

-- PHP → Lógica do servidor e integração com o banco de dados.
-- MySQL → Armazenamento das mensagens de contato.
-- HTML5 & CSS3 → Estrutura e estilos da aplicação.
